# Driver Drowsiness Detection

## Project Overview
This project detects driver drowsiness in real time using computer vision and deep learning. The system monitors eye movements, detects prolonged eye closure, and triggers an alert to help prevent road accidents.

## Technologies Used
- Python
- OpenCV
- TensorFlow
- Keras
- NumPy
- Computer Vision

## Features
- Real-time face detection
- Eye detection and monitoring
- Drowsiness classification
- Audio alert system
- Performance evaluation using deep learning models

## Files Included
- Main application code
- Model training notebooks
- Haar Cascade classifiers
- Requirements file
- Demo GIF
- Performance graphs

## Project Outcome
The system provides an early warning mechanism for fatigued drivers, helping improve road safety and reduce accident risks.
